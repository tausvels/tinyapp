## Dependencies

- Node.js
- Express ^4.17.1
- EJS ^3.0.1
- bcrypt ^2.0.0
- body-parser ^^1.19.0
- cookie-session ^1.3.3
- mocha ^ 6.2.2
- chai ^4.2.0
- grpc ^1.24.2
- nodemon ^2.0.2

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
